# OpenRoot Utility

Утилита для автоматизации открытия root-доступа на тестовых терминалах Vinteo.

## Возможности

* Подключение под support-пользователем
* Установка пароля root
* Включение PermitRootLogin
* Перезапуск SSH

## Требования

* expect
* ssh

## Запуск

```bash
chmod +x openroot
./openroot
```
